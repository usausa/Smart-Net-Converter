# Smart.Converter .NET - Type converter library

[![NuGet](https://img.shields.io/nuget/v/Usa.Smart.Converter.svg)](https://www.nuget.org/packages/Usa.Smart.Converter)

## Features

* `IObjectConverter` / `ObjectConverter` — type conversion system with factory-based extensibility
* Built-in converter factories for numeric types, `bool`, `string`, `Enum`, `DateTime`, `Guid`, `DBNull`, collections, and more
* Supports cast operators, constructors, and parse methods for conversion
